# Salma Mulla-Supe — Portfolio Website

**Pages**
- Introduction (homepage)
- Media (video)
- Content (LinkedIn feed)
- Articles (DEV.to integration)
- Contact

**Homepage**
- Profile photo with "Carpe Diem" speech bubble (quote rotates every 24 hours between 3 options)
- Animated hero headline with word-by-word reveal
- Stats row: 20+ years experience, CDPSE certified, Deloitte Senior Manager
- Professional credentials and certifications card
- Career timeline (Deloitte Qatar, Deloitte India, Mphasis)
- Awards and recognition section
- Floating LinkedIn recommendation quotes on screen edges, cycling as you scroll
- Light/dark mode toggle (preference saved)
- Scroll progress bar

**Media Page**
- Embedded video player with play/pause and mute controls
- Fallback link to LinkedIn post if video unavailable

**Articles Page**
- Live pull from DEV.to, auto-updates when new articles are published
- "Coming Soon" placeholders until first article is posted

**Contact Page**
- Email, phone, location
- Direct links to LinkedIn, X/Twitter (@iconceptualist), Instagram (@salmasupe)

**General**
- Fully responsive
- Consistent branding: Salma Mulla-Supe across all pages
- Deployed to Cloudflare Pages
